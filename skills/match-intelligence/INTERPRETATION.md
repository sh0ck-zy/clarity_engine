# Reading the Data

Quick reference for interpreting our tools' output.

## Form Strings

```
WWWWW = Dominant, 15/15 points
WDWDW = Solid, 11/15 points  
DDDDD = Drawing everything, 5/15 points
LLLLL = In crisis, 0/15 points
```

**Direction matters more than total:**
- `LLWWW` → Improving, momentum building
- `WWWLL` → Declining, something's wrong

## xG (Expected Goals)

**Per game benchmarks:**
- \>2.0 xG/game = Elite attack
- 1.5-2.0 = Good
- 1.0-1.5 = Average
- <1.0 = Struggling

**Goals ÷ xG = Finishing efficiency:**
- \>1.2 = Clinical (or lucky, will regress)
- ~1.0 = Converting as expected
- <0.8 = Wasteful (or unlucky, might improve)

## What "Good Form" Actually Means

Don't just count Ws:
- WDWDW against top 6 = excellent
- WWWWW against bottom 5 = means less
- 4 straight 1-0 wins = fragile
- 4 straight 3-1 wins = dominant

Context matters more than the letter.
