## Codebase Patterns
- Ralph agent artifacts live under `ralph/` (prd.json and progress.txt).
- **Validation code lives under `src/validation/`** (NOT ralph/validation!) and uses dataclasses for report schema.
- **Tests live under `tests/`** (NOT ralph/test_*.py!)
- Database schema lives in `src/database/schema.sql`; scripts follow the `PROJECT_ROOT` + `src` path injection pattern.
- **IMPORTANT: Never create application code under `ralph/` - only orchestration files (PRD, progress, prompts).**
